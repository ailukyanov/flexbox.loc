What is Flexbox?

Flexbox is a whole module that contains a set of properties and their values for both the parent of a flex container and its children of flex items.

The main advantage of flexible FLEX layouts is the ability to change the width / height of the space around and the order of flex items in order to best fill the available space of the flex container.
